How To Print 3D Objects

It is quite simple to print out a 3D object! All you must do is follow the steps listed below and your object will be printed as soon as possible.

1. Design your object using your choice of software (CAD, Google SketchUP, etc) and save as a .stl file

2. Import your .stl file to a conversion software (Simplfy3D, Slic3r, etc) and convert it to a gcode file

3. Fill out the required fields under the Order tab

4. Upload your gcode file

5. Hit submit and you are done!
